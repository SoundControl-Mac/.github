## 💡 Overview

<a href="#soundControl">SoundControl</a> solves one of macOS's most persistent audio limitations — the operating system's single system-wide volume control that affects every application simultaneously, making it impossible to keep music quiet while a video call stays loud, or to mute notification sounds without muting everything else.

The <a href="#soundControl">per-application volume control</a> is SoundControl's core capability — each running application that produces audio gets its own independent volume slider, accessible from the menu bar without opening individual application preferences or fumbling with system-wide volume that affects everything at once. Lowering Spotify's volume while keeping Zoom at full volume, muting Slack notification sounds while music continues playing, or balancing multiple simultaneous audio sources — these become simple slider adjustments rather than constant manual juggling of the single system volume control that macOS provides by default. The <a href="#soundControl">menu bar quick access</a> places all running applications' volume controls in a single dropdown accessible from the menu bar — clicking the SoundControl icon reveals every audio-producing application with its current volume level and a slider for immediate adjustment, without navigating to individual application windows or system preferences.

The <a href="#soundControl">output device switching</a> changes the active audio output device directly from the menu bar — switching between built-in speakers, headphones, AirPods, and external audio interfaces without navigating System Settings. This becomes particularly valuable for users who regularly switch between multiple audio output devices throughout their work. The <a href="#soundControl">per-app output routing</a> in advanced configurations sends different applications to different output devices simultaneously — music to speakers while call audio routes to headphones, a capability that standard macOS audio routing doesn't provide without complex third-party audio routing software.</p>

<a href="#soundControl">Customizable keyboard shortcuts</a> configure hotkeys for common volume operations — adjusting the volume of the frontmost or last-used application without opening the menu bar dropdown, muting all audio instantly, or cycling through output devices via keyboard. <a href="#soundControl">Visual volume indicators</a> show real-time audio level activity for each application — a visual cue for which applications are currently producing sound, useful for identifying the source of unexpected audio or background noise.

<a href="#soundControl">Persistent volume memory</a> remembers individual volume settings for each application between sessions — an application set to 40% volume stays at 40% the next time it launches and produces audio, rather than resetting to full volume and requiring re-adjustment. <a href="#soundControl">Lightweight menu bar operation</a> runs continuously with minimal system resource usage, providing constant audio control access without a persistent dock icon or window taking up screen space.</p>
